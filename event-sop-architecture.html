01
企劃
3 項
活動大表 / 行事曆建立
統整年度所有活動時程，建立可追蹤的主行事曆，確保跨活動不衝突、資源可調配。
活動策略 / 企劃表
定義單場活動目標、受眾、KPI、預算框架與核心訊息，作為所有執行工作的依據文件。
風險預案 / 應變計畫
預擬講者取消、場地異常、天候影響等情境的 Plan B，確保活動韌性。
02
議程
7 項
[比價] 主持人安排
依據活動調性搜尋、比價並確認專業主持人，含中英雙語能力評估。
主持人講稿、內容授權 / 審核流程
撰寫主持人串場稿，並建立講者簡報的合規審核流程（尤其金融機構有 compliance 要求）。
活動流程表
完整時間軸 rundown，含每一環節起訖時間、負責人、轉場提示與技術 cue 點。
問答題目準備 / Panel 設計
為圓桌論壇或 Q&A 環節預擬引導問題，設計互動節奏與議題深度層次。
機構 / 講者邀請與確認
向目標機構與講者發出正式邀約，追蹤回覆狀態直到確認出席。
講者 rundown、投影片準備 / 索取
向已確認講者索取簡報檔案，統一格式與版型，並提供個人化 rundown。
講者資料收集（bio / 照片）
收集講者中英文簡歷、高解析度照片與職稱，供網站、手冊及現場識別使用。
03
來賓
5 項
每週定期名單回顧
每週固定追蹤報名狀態、出席率預估與邀請缺口，及時調整推進策略。
活動報名表
設計並上線報名表單，收集與會者基本資料、特殊需求（飲食禁忌、翻譯等）。
會前出席確認 / 接待安排 / VIP 分級策略
會前逐一確認出席，依理事、會員、外部貴賓分級規劃接待規格與動線。
邀請名單策略討論
與主管及 BD 團隊共同定義目標受眾輪廓與名單來源，確保邀請品質。
邀請名單 push / 追蹤
持續推進未回覆名單，透過多管道（電話、LINE、Email）觸及並記錄回覆狀態。
04
行銷
8 項
文案撰寫（網站 / 社群）
產出活動主文案與各渠道適配版本，維持品牌調性與高淨值受眾語感。
社群曝光活動
依通路特性（官網 / EDM / LINE / WeChat / LinkedIn / FB）發布活動資訊與宣傳素材。
活動小結公告社群
活動結束後產出精華摘要，於全渠道同步發布，強化品牌影響力與社群記憶。
活動花絮上架網頁、社群露出
整理活動照片、短影片與金句圖卡，上架官網並於社群持續露出，延長內容生命週期。
活動報名表
報名頁上線與轉換數據追蹤，監測各渠道帶入的報名數量與轉換率。
活動資訊上架網頁 Google Site
將活動完整資訊（議程、講者、報名連結）上架官方網站，確保資訊一致。
倒數提醒（D-7 / D-3 / D-1）
於活動前固定節點推送提醒訊息，降低 no-show 率並營造期待感。
新聞稿 / 媒體合作 / PR
撰寫新聞稿、洽談媒體露出機會，建立並維護媒體聯絡名單。
05
合作 / 贊助
6 項
合作方案洽談 / 設計
規劃合作權益結構，與潛在夥伴進行需求對接並擬定合作框架。
合約簽署
確認合作條款後推動合約用印流程，確保雙方權責清晰。
贊助方案設計
設計多層級贊助方案（冠名 / 協辦 / 支持），明確各級露出權益與交付物。
贊助交付（票券 / 名單）
依合約提供贊助商應得權益，包含免費票券、與會者名單或展位等。
贊助成果報告
活動後彙整贊助商露出數據、觸及效果與照片，交付正式成果報告。
KOL / 夥伴曝光
協調合作夥伴或意見領袖進行活動前後的社群轉發與背書。
06
場地
5 項
[比價] 場地 / 貴賓室搜尋與比較
依活動規模、品牌調性與預算搜尋場地，製作比較表供決策。
[比價] 燈光、攝影 / 錄影、AV 設備 / 直播 / 網路
評估並比價視聽設備供應商（投影、音響、麥克風、直播串流、WiFi 增強等）。
交通動線 / 停車 / 接駁資訊準備
整理場地周邊交通、停車場與接駁安排資訊，製作與會者指引。
場地及相關設備預訂
完成場地合約簽訂、設備租借訂單，並鎖定確認日期與撤場時間。
場地佈置、配置圖 / 座位安排
繪製平面配置圖，規劃座位區、報到區、VIP 區與動線，確認佈置時程。
07
設計
3 項
主視覺、社群素材、活動頁設計
開發活動主視覺 KV 並延伸為各渠道素材（社群圖、Banner、活動頁面）。
活動簡報
製作活動介紹用簡報，供對外洽談合作、招募贊助或內部提案使用。
識別證、背板、易拉展、現場指引看板
設計並送印現場所有實體視覺物件，確保品牌一致性與空間指引清晰。
08
執行
6 項
[場佈] 輸出、攝影、設備、投影 / 錄影
活動前完成所有印刷品輸出、攝影設備到位、投影與錄影系統測試。
活動前彩排
安排講者與工作團隊現場走位，確認設備、燈光、音響與轉場流程。
現場貴賓接待
依 VIP 分級安排專人迎接、引導入座與休息室服務，展現高規格待客。
細流 / 工作人員分工
產出分鐘級細部流程表，明確每位工作人員的崗位、職責與通訊機制。
報到流程 / 簽到表 / QR
建置報到系統（QR Code 掃描或人工簽到），確保入場順暢並即時統計出席。
攝影、大合照安排
規劃活動攝影動線與大合照時機，確保關鍵畫面不遺漏、照片可即時交付。
09
財務
5 項
成本控管
持續追蹤實際支出 vs 預算，即時示警超支項目並調整配置。
活動費用結清、預扣費用追回
活動結束後完成所有款項結清，追回預付保證金或未使用之預扣款。
活動預算表
編列完整預算明細（場地、設備、餐飲、印刷、講者費等），經核准後作為控管基準。
報帳、發票
彙整所有費用憑證與發票，依公司財務流程完成報帳作業。
講者 / 合作方付款時程追蹤
追蹤講者酬勞與廠商尾款的付款進度，避免延遲影響合作關係。
10
飲食
3 項
[比價] 餐廳搜尋與比較
依預算、菜系與場地距離篩選餐廳，製作比較表（含菜單、人均價位、包廂條件）。
餐廳 / 餐點預訂
確認最終人數與特殊飲食需求後完成訂位，並確認出菜時間與服務流程。
禮品採購
規劃講者伴手禮或與會者紀念品，統一採購並安排包裝與發放。
11
會後
3 項
活動復盤、結案會議
召開內部復盤會議，檢視 KPI 達成度、記錄優化項目，產出結案報告。
活動滿意度調查問卷
於活動後 48 小時內發送問卷，收集與會者回饋作為下次迭代依據。
潛在會員名單更新、名片生態圈更新
將活動中接觸的潛在會員匯入 CRM，更新名片資料庫並移交 BD 團隊跟進。
12
海外
6 項
交通（機票 / 巴士）
統籌團體機票訂購與當地交通（巴士包車 / 專車），管理航班與轉機銜接。
行程路線規劃、每日 Rundown、機構接待時程
編排每日參訪路線與時程，與各接待機構確認行程細節並預留緩衝時間。
就醫、緊急聯絡機制
建立當地緊急聯絡清單（醫院、大使館、保險公司），確保突發狀況可即時應對。
當地接待、翻譯
安排當地接待人員與專業翻譯，確保語言與文化溝通無障礙。
機構單位安排
與海外目標機構進行邀約、行程確認與接待細節對接，確保參訪品質。
簽證 / 保險
協助團員確認簽證需求、統一辦理旅平險，並建立團員護照資料清冊。
